# 命名建議器

根據用戶描述，生成符合命名規範的名稱建議。

## 使用方式

```bash
/suggest-name <描述>
```

**範例**：
```bash
/suggest-name 計算使用者購物車總金額的函數
/suggest-name 儲存用戶登入狀態的變數
/suggest-name 處理 API 錯誤的類別
```

---

## 執行步驟

1. 分析用戶描述，判斷類型（函數/變數/類別/常數）
2. 使用 `naming-conventions` Skill 的規範生成名稱
3. 輸出建議名稱

---

## 輸出格式

```
## 命名建議

**描述**: <用戶輸入的描述>
**類型**: 函數 / 變數 / 類別 / 常數

### 建議名稱

1. `name1` - 說明
2. `name2` - 說明
3. `name3` - 說明

### 推薦

`推薦名稱` - 推薦原因
```
